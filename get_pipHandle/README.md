该工程实现的功能：
   
   查询USB设备端口信息，创建批量传输管道句柄，构造批量传输IRP，发送到目标设备中

数据发送：
USB四种传输模式

1、Bulk传输事务
大块的数据传输，Bulk管道。
2、控制传输事务
向USB设置一些控制信息，控制管道。
3、中断传输事务
传输的数据量很小，一般用于通知Host某个事件的来临。
4、同步传输事务
数据的同步性，传输视频数据的时候会采用这种事务。
